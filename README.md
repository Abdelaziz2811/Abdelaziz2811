### 🚗 DVLD - Driving License Management System

![C#](https://img.shields.io/badge/C%23-239120?style=flat&logo=c-sharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-512BD4?style=flat&logo=dotnet&logoColor=white)
![SQL Server](https://img.shields.io/badge/SQL_Server-CC2927?style=flat&logo=microsoft-sql-server&logoColor=white)
![ADO.NET](https://img.shields.io/badge/ADO.NET-512BD4?style=flat&logo=dotnet&logoColor=white)

**⭐ My Most Complex Project** - A full-featured WinForms application simulating a real government licensing department with proper 3-tier architecture.

**🎯 Core Features:**
- 🆕 **New License Issuance** (multiple categories)
- 🔄 **License Renewal** with validation
- 📝 **Re-Examination Service** (vision, theory, practical)
- 🔒 **Replacement Services** (lost/damaged/detained)
- 🌍 **International License** issuance
- 👥 **Applicant Management** (unique records, full details)
- 📋 **Application Tracking** (duplicate prevention)
- ✅ **3-Step Examination** (vision → theory → practical)
- 🔐 **User Management** with audit trail

**🏗️ Architecture:**
- **3-Tier Design** (Presentation → Business Logic → Data Access)
- **ADO.NET** for efficient database operations
- **SQL Server** backend with complex relationships
- **Business Rules Engine** (age validation, prerequisites, fees)

**📊 Technical Complexity:**
- 8+ license categories with different rules
- Multi-step workflows with state management
- Complex validations (age, previous licenses, tests)
- Audit trail for all operations
- Comprehensive CRUD on multiple entities

**Development Period:** May - July 2025
